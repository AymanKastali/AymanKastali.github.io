Project Brief

Job Search is the best website to you if you are looking for a skillfull employees to fill your jobs positions, or if you are looking for a great job matching your skills.
Also you can to create a new connections and interact with other users and companies by sending connection request, follow pages, using chat system, posting posts, comment, reply and like.
For employers you can filter candidates for job posts, organize them and send email with information of interviews and acceptance of a specific job post, also a helpfull tool to use when making interviews.
And for employyes a job positions recommender allows you to see job posts positions matching your needs, and a filter search for posts.

Features

  • - Django allauth package for handling user authentication.

  • - Django friendship package for handling friends requests, follows and blocks.

  • - Beautiful GUIs using css grid system besids bootstrap.

  • - Python decorators for security and managing pages access.

  • - Email sender feature between companies and candidates.

  • - Filter jobs by tag, city and job title for users who are looking for a job.

  • - Super helpful page for HR to organize candidates, taking notes while interviewing and accpect the best candidate.

  • - Real time chat system.

Used Tools

  • Django framework
  • PostgreSQL
  • HTML, CSS, Javascript
  • Bootstrap
  • Django allauth for authentication
  • Django Channels
  • Django friendship package
